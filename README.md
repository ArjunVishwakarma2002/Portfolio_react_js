# Personal Portfolio Website

Modern MERN Stack Portfolio Website built with React.js, Node.js, Express.js, and MongoDB showcasing projects, technical skills, certifications, and responsive UI design.

---

## 🚀 Features

- Responsive Portfolio Website
- Modern UI Design
- About Section
- Skills Section
- Projects Showcase
- Resume Download
- Contact Form
- MongoDB Database Integration
- REST API Integration
- React Frontend
- Express & Node.js Backend

---

## 🛠️ Tech Stack

### Frontend
- React.js
- HTML5
- CSS3
- JavaScript
- Axios

### Backend
- Node.js
- Express.js

### Database
- MongoDB
- Mongoose
